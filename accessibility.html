Accessibility Statement

Big 7 Construction is committed to making this website usable by as many people as possible, including people with disabilities. This page explains what we are doing and how to reach us if any part of the site is not working for you.

Our commitment

We aim to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA. These guidelines cover readable text, sufficient color contrast, keyboard navigation, and clear labeling — all of which help make the site work for people using screen readers, screen magnifiers, or keyboard-only navigation.

What we have done

  • Descriptive alt text on meaningful project images.
  • A visible keyboard focus indicator throughout the site.
  • A skip-to-content link at the top of every page for keyboard users.
  • A single <main> landmark on every page for straightforward screen-reader navigation.
  • Proper labels on every input on the quote intake form.
  • Named buttons for icon-only controls such as the mobile menu.
  • Support for prefers-reduced-motion so users who ask for less animation see less.
  • A defined page language (lang="en") so screen readers pronounce content correctly.

Known limitations

Accessibility is ongoing work. If you find something that gets in your way — a page you can't navigate, text you can't read, a form that won't accept your input — please tell us. We treat accessibility reports as a priority and address them as quickly as we can.
